# LeadFlow Backend

Node.js + Express REST API for the LeadFlow monorepo.

## Setup

```bash
cd backend
cp .env.example .env
npm install
npm run dev
```

## Available Scripts

| Script | Description |
|---|---|
| `npm run dev` | Start dev server with nodemon |
| `npm start` | Start production server |

## API Endpoints

| Method | Route | Description |
|---|---|---|
| GET | `/api/health` | Health check |
